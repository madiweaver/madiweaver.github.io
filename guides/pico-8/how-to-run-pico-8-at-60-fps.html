How to Run PICO-8 at 60FPS

19, November 2022

By default, PICO-8 will run your program at 30fps (30 frames per second). If you would like to run your program at 60fps, simply rename your _UPDATE() function to _UPDATE60().

FUNCTION _UPDATE60()
  -- YOUR UPDATE LOGIC HERE
END
